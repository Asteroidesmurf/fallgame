# Canvas Fallgame ES6 canvas version
Canvas fallgame based upon [BrandedGames](http://fallgame.brandedgames.nl/).

## Installation
First install dependencies:
```
npm install
```

Then compile the assets using webpack:
```
npm run dev
```

And open index.html in the browser! :D

## Browsersync
To make webpack watch the files and start a browsersync server. Run:
```
npm run watch
``` 

The server starts on `http://localhost:3000`.
